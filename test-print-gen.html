Calibration Sequence

Calibration is most effective when done in order. Each step affects the next:

  1. Bed leveling / first layer — everything else depends on good bed adhesion
  2. E-steps — correct extrusion amount before tuning flow or temperature
  3. Temperature tower — optimal temp for the rest of calibration
  4. Flow rate / extrusion multiplier — fine-tune after E-steps
  5. Retraction — tune after flow so stringing isn't confused with under-extrusion
  6. Pressure Advance / Linear Advance — final fine-tuning
  7. Speed / max flow — push limits only after baseline is solid
